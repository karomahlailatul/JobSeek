# JobSeek 

## About the project
JobSeek Built up with NextJs with backend ExpressJS and deployment on Vercel.

## Usage 

### Clone
```shell
git clone https://github.com/karomahlailatul/JobSeek.git
```
### Installing 
```shell
npm i;
```

### Testing
```shell
npm run start;
```
